# DISCORD BOT for Stock and Crypto

This is a discord bot that gives real live data of
*Weather
*Stock Info
*Crypto Info
*Lists Top Crypto datas

Additional Features:
*Gives Schedule of AWS Loft
*Recognizes a photo for person/object
*Tells jokes
